# Cognifyz Data Science Internship

This repository contains the work completed as part of the Data Science Internship at Cognifyz Technologies.

## Overview
The internship focused on applying data science concepts using Python to analyze real-world datasets, extract insights, and build basic predictive models.

## Tasks Completed

### Level 1: Data Exploration & Analysis
- Data loading and preprocessing
- Handling missing values
- Descriptive analysis
- Geospatial analysis
- Data visualization

### Level 2: Business Insights & Feature Engineering
- Table booking and online delivery analysis
- Price range analysis
- Feature engineering
- Insight generation using visualizations

### Level 3: Advanced Analysis & Modeling
- Predictive modeling using Linear Regression
- Customer preference analysis
- Advanced data visualization

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Files
- `Level_1.ipynb` – Data exploration and descriptive analysis  
- `Level_2.ipynb` – Business analysis and feature engineering  
- `Level_3.ipynb` – Predictive modeling and visualization  
- `Dataset.csv` – Dataset used for analysis
